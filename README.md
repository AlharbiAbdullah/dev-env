# work_setup

My dev environment configs. Clone and run the setup scripts on a fresh machine.

## What's inside

- `terminal/` - zsh, WezTerm, tmux configs + installer
- `wm/` - i3 window manager config (mapped from Aerospace) + installer

## Usage

```bash
git clone https://github.com/AlharbiAbdullah/work_setup.git ~/work_setup

# Terminal
cd ~/work_setup/terminal && ./setup.sh

# i3 (Linux only)
cd ~/work_setup/wm && ./i3_setup.sh
```
